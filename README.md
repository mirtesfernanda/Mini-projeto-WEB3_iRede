
# Mini Projeto Web3 - Registro de Status

Projeto Web3 funcional que registra status na blockchain.

## Contrato utilizado
0x2D7C9F01c281Ab8A72e703e9E150dbe76C63B4f3

## Funcionalidade
- Conectar carteira Metamask
- Enviar novo status
- Registrar transação na blockchain
- Exibir hash da transação

## Tecnologias
Solidity
Metamask
Ethers.js v6
HTML CSS JavaScript

## Como usar
1. Abrir index.html
2. Conectar Metamask na rede Sepolia
3. Digitar status
4. Clicar atualizar
