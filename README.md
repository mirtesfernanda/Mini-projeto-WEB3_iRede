
# Mini Projeto Web3 - Registro de Status

Projeto Web3 funcional que registra status na blockchain.

## Contrato utilizado
0x8EfeFd67F502B4Fd37D5C102E60cE1e07123baa9

## Funcionalidade
- Conectar carteira Metamask
- Enviar novo status
- Registrar transação na blockchain
- Exibir hash da transação

## Tecnologias
Solidity
Metamask
Ethers.js v6
HTML CSS JavaScript

## Como usar
1. Abrir index.html
2. Conectar Metamask na rede Sepolia
3. Digitar status
4. Clicar atualizar
